## Installation

The library is available through [CocoaPods](http://cocoapods.org). To install it, simply add one of the following lines to your Podfile.

Swift 3:

```ruby
pod "AppConfigSwift", '~> 0.7.2'
```

Swift 2.2:

```ruby
pod "AppConfigSwift", '0.7.0'
```
